In this lab, I’m building a simple web page and practicing the most common HTML tags used in real websites.

My goal is to understand how structure works first, then I can style the page later using CSS.

HTML structure keeps content organized, and clean code is easier to read. I can add tiny notes like 1 or formulas like H2O. This text is wrapped in a span for inline control.

Links connect pages, and good navigation helps users move around. For example, x2 is a power, and CO2 is a common chemical formula. Span is useful when I want to style only part of a sentence later.

Images make pages more visual, and alt text improves accessibility. I might write 10th place or use small indices like ai. Span is inline, so it doesn’t break the line like a div.

Paragraphs split ideas, and emphasis shows what matters. I can mark a footnote * and show a sub value like n0. Span helps highlight a word without changing the whole paragraph.

Consistency matters: each paragraph must include all required tags. I can write E = mc2 and also show log10. This final span confirms the requirement is satisfied.
